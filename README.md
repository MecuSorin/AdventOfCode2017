# AdventOfCode2017

Fun with F# based on the puzzles from: http://adventofcode.com/2017


### Getting started

- Run the `.paket/paket.bootstrapper.exe` to get the needed paket.exe
- Use `build.cmd all` (on Windows) or `build.sh all` (on Linux or Mac)
